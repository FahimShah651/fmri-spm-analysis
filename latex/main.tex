\section{Datasets and Experimental Design}
\label{sec:datasets}

Two publicly available datasets from the OpenNeuro repository were
used in this project, chosen to cover distinct functional systems.

\subsection{Dataset 1 — ds000114 (Motor Task)}
The \textit{test-retest fMRI dataset}~\cite{Gorgolewski2013}
(ds000114) includes a blocked motor task in which subjects perform
repetitive finger tapping, foot tapping, and lip movements, each
interleaved with rest. Subject \texttt{sub-01}, session
\texttt{ses-test}, run~1 was analysed (184 volumes). The T1
anatomical image from the same session was used for normalisation.
Scan parameters read from the NIfTI header were used directly in
the preprocessing scripts.

\subsection{Dataset 2 — ds000105 (Visual Task)}
The \textit{visual object recognition} dataset~\cite{Haxby2001}
(ds000105) presents eight visual categories — faces, houses, cats,
bottles, scissors, shoes, chairs, and scrambled images — in a
blocked design. Subject \texttt{sub-1}, run~1 was analysed (121
volumes). This dataset is widely used in the literature as a benchmark
for multivariate pattern analysis; running a standard GLM on it
provides an immediate sanity check against published results.

\section{Discussion}
\label{sec:discussion}

The results across both datasets were broadly consistent with
expectations from the neuroimaging literature, which suggests the
pipeline is working correctly.

\textbf{Motion quality.}
Mean framewise displacement was below 0.2\,mm in both scans, putting
them in the ``low motion'' category under the criteria of Power
et~al.~\cite{Power2012}. This matters because even small amounts of
systematic motion — particularly correlated with the task — can
produce spurious activations that mimic genuine BOLD responses. The
clean motion profiles here make it unlikely that the activation maps
are motion artefacts.

\textbf{Motor cortex somatotopy.}
The three finger/foot/lips contrasts in Dataset~1 produced activation
peaks that followed the expected somatotopic organisation of primary
motor cortex from medial (foot) to lateral (face/lips)
territory~\cite{Lotze2000}. The omnibus AllMotor map covered the
entire motor strip plus the cerebellum and thalamus — the core nodes
of the voluntary motor system. These results validate both the
preprocessing (correct spatial normalisation) and the GLM specification
(correct event timing).

\textbf{Visual category selectivity.}
The face-selective fusiform cluster in Dataset~2 (Fig.~\ref{fig:ds2:tmap:face})
matches the reported location of the fusiform face area in the
original Haxby et~al.\ paper~\cite{Haxby2001}. Replicating a
published finding using an independent implementation of the same
pipeline is a strong form of validation. The other category contrasts
produced distributed activation in the lateral occipito-temporal
cortex, consistent with object-selective processing in the ventral
stream.

\textbf{Functional connectivity.}
Both connectivity maps showed spatially coherent networks. The motor
seed produced bilateral M1 and SMA connectivity (the sensorimotor
network), and the visual seed produced occipito-temporal connectivity
(the visual network). The cross-dataset comparison in
Fig.~\ref{fig:comparison:connectivity} shows these two networks are
spatially non-overlapping, which is what we would expect from
independent functional systems. The negative correlations with
default-mode regions in both maps are also expected: task-positive
networks tend to be anti-correlated with the default mode
network~\cite{Biswal1995}.

\textbf{Limitations.}
This analysis covers a single subject from each dataset, so the
results cannot be generalised beyond these two individuals. The
connectivity analysis was performed on the full task run (not a
separate resting-state scan), so the correlation structure partially
reflects task-related coactivation rather than purely intrinsic
connectivity. Future work should include group-level analyses and
separate resting-state acquisitions.
